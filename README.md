# Matrimonial Application (Java)

## Description
A console-based Matrimonial Application developed using Core Java.
The application evaluates user profile eligibility using nested
conditional statements based on real-world criteria.

## Features
- User input handling using Scanner
- Asset and salary based eligibility check
- Age, height, weight validation
- Logical flow using nested if-else statements
- Console-based interaction

## Concepts Used
- Core Java
- Nested if-else statements
- Scanner Class
- Console I/O
